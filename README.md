# E-Commerce-MERN
E-commerce website built using ReactJS, NodeJS, ExpressJS and MongoDB

 ## Functionalities
1.  User authentication
2.  Admin authentication
3.  Product CRUD 
4.  Category CRUD
5.  Order CRUD
6.  Managing user orders
